API Documentation

Welcome to the ConvertPro API. Our RESTful API allows you to programmatically manage file conversions, user permissions, and analytics reports.

link Base URL

https://api.convertpro.io/v2

lock Authentication

Authenticate your requests by including your API key in the Authorization header.

Example Header content_copy
curl -X GET "https://api.convertpro.io/v2/user" \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json"

Access Roles

admin_panel_settings

Admin

Full Access

Manage API keys, billing, and team members across the organization.

developer_mode

Developer

Write Access

Execute conversions and manage webhooks. No billing access.

visibility

Viewer

Read Only

View job status and download converted reports. Cannot initiate new jobs.
