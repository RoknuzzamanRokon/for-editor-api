ConvertPro API Documentation

Welcome to the ConvertPro API. Our API allows you to programmatically convert documents between hundreds of formats, manage user accounts, and track conversion usage in real-time.

Production Base URL

https://api.convertpro.io/v2

Authentication

The ConvertPro API uses JWT (JSON Web Tokens) for authentication. To access protected endpoints, you must include a valid bearer token in your request headers.

POST /api/v2/auth/login 
curl -X POST https://api.convertpro.io/v2/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "api_key": "your_api_key_here",
    "secret": "your_secret_hash"
  }'

User Roles

Super User

Full access to system configuration and billing.

manage_accounts

Admin

Manage organization users and view conversion logs.

person

General

Default role. Can perform conversions and view own history.

visibility

Demo

Read-only access with restricted conversion limits.
