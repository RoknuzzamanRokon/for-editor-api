PDF to DOCX

POST /v1/convert/pdf-docx

Cost per conversion 3 Points

Description

High-fidelity conversion from PDF to Microsoft Word (DOCX) format. Supports OCR for scanned documents and preserves layout, images, and font styles.

Headers

Authorization: Bearer YOUR_API_KEY
Content-Type: multipart/form-data
X-Callback-URL: https://your-webhook.com

Code Snippet

const axios = require('axios');
const fs = require('fs');

const form = new FormData();
form.append('file', fs.createReadStream('doc.pdf'));

axios.post('https://api.convertpro.com/v1/convert/pdf-docx', 
  form, {
  headers: { 'Authorization': 'Bearer KEY' }
}).then(res => console.log(res.data));
